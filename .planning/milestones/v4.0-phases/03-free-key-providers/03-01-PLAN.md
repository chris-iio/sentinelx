- Class URLhausAdapter with name="URLhaus", requires_api_key=True
- supported_types = frozenset({IOCType.URL, IOCType.IPV4, IOCType.IPV6, IOCType.DOMAIN, IOCType.MD5, IOCType.SHA256})

- ENDPOINT_MAP dict mapping IOCType -> (url_path, body_key) tuple:

  1. Check ioc.type in supported_types, return EnrichmentError if not

  6. body = read_limited(resp)
  7. Parse verdict from query_status field:
     - "is_listed" (URL endpoint) -> "malicious"
     - "ok" with urls_count > 0 (host endpoint) -> "malicious"
     - "ok" with payloads (payload endpoint) -> "malicious"
     - "no_results" or "no_result" -> "no_data"
  8. Return EnrichmentResult with raw_stats
- Module-level _parse_response() function (stateless, takes ioc, body, provider_name)
- Exception handling: Timeout -> EnrichmentError, HTTPError -> EnrichmentError, general Exception -> log + EnrichmentError
